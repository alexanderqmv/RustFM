## Build from source
###### 1. Clone Repo
```shell
git clone https://github.com/alexanderqmv/rs-filemanager.git
```
###### 2. Cargo Build
```shell
cargo build
```
###### 3. Cargo Run
```shell
cargo run
```


### You will see:
```shell
$ Usage:  <command> <path> <data>
Commands:
        fwrite
        fread
        ffind
        fcreate
        fremove

```
